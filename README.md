🍎 Food Market Analysis
End-to-End Data Analytics Project (Excel + SQL + Power BI)

📌 Project Overview
This project analyzes a retail food market dataset to uncover profitability drivers, promotion effectiveness, customer behavior patterns, and store performance metrics.
The project follows a complete data analytics workflow:
Excel → Data Cleaning & Preparation
SQL (MySQL) → Business Analysis & KPI Calculation
Power BI → Dashboard Creation & Visualization
The goal is to transform raw retail data into actionable business insights.

🔄 Project Workflow 
🟢 1️⃣ Data Cleaning – Excel
Removed duplicates
Handled missing values
Standardized column formats
Ensured consistent numeric formatting
Prepared dataset for SQL import

🔵 2️⃣ Data Analysis – SQL
Performed advanced business analysis including:
Profit Margin Calculation
ROI (Return on Investment) Analysis
Correlation Analysis
Customer Segmentation
Revenue per Square Foot
Promotion Performance Evaluation
Income Group Analysis

🟣 3️⃣ Data Visualization – Power BI
Built an interactive dashboard to visualize:
Profit by Food Category
Promotion ROI Comparison
Membership-Level Profit Contribution
Revenue per Sqft by Store Type
Income Group Purchasing Patterns
Media Type Performance
The dashboard enables business stakeholders to quickly identify trends and profitability drivers.

🎯 Key Business Questions Answered
Which food categories generate the highest profit margin?
Do promotions generate positive ROI?
Is price per unit correlated with unit sales?
Do higher membership levels generate more profit?
Does income level influence purchasing power?
Are low-fat products more profitable?
Do stores with larger grocery ratios perform better?
Which store type generates the highest revenue per sqft?
Does education level impact purchasing behavior?
Which media type provides the best profit per promotion cost?
📊 Key Insights
Meat category shows the highest profit margin (60%).
Most promotions generate negative ROI.
Price per unit has almost no correlation with unit sales (-0.002).
Bronze members contribute the highest total profit.
Medium-income customers generate the highest overall profit.
Gourmet Supermarkets generate the highest revenue per square foot.
Daily Paper and TV media provide the highest profit efficiency among promotion channels.
📈 Business Recommendations
Focus on high-margin categories to improve profitability.
Re-evaluate promotion strategies to reduce negative ROI.
Optimize marketing budget toward high-performing media channels.
Use customer segmentation instead of relying only on price adjustments.
Improve store layout and product mix to maximize revenue per sqft.
